建立資料夾結構

為了讓專案更有條理、方便後續維護，我們在 src 底下建立以下資料夾：

  • config：存放設定檔（例如資料庫連線設定）。
  • controllers：處理 API 業務邏輯。
  • entities：定義資料庫模型 (Entity)。
  • middleware：存放中間件。
  • routes：定義 API 路由。
  • utils：常用工具函數。
